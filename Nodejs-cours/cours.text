NodeJs est ni un language de programmation, ni un framework mais un envirenement pour ceci. 
Node est ercit en plusieurs languags; (Javascript(le compilateur de package)), C et C++ (runtime pour executer le js est ecrit en c++)
NodeJs est utilisé en frontend et backend; il n'est pas utilisé que pour construire le serveur. 
Javascript est un language commun utilisé pour faire à la fois le back et le front end à l'aide de framework.
Creer une fonction et si jamais on aura utiliser dutilise cette fonction on peut l'utiliser dans les deuc coté 
Cela permet de gagner en productivité et efficacité
